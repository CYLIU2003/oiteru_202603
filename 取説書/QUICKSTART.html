OITERU かんたんスタートガイド

標準構成は Linux 系 OS + tmux + ローカル MySQL + db_server.py + unit.py です。Docker は標準手順では使いません。

必要なソフトウェア

ソフトウェア用途URL
Ubuntu親機 Linux OS の候補ubuntu.com/download
Raspberry Pi Imager子機 OS 書き込みraspberrypi.com/software
Raspberry Pi OS子機 OSoperating-systems
Python実行環境python.org/downloads
Gitバージョン管理git-scm.com/downloads
MySQL Community Server親機 DBdev.mysql.com/downloads/mysql
tmux起動管理tmux releases
Visual Studio Code推奨エディタcode.visualstudio.com/download
Tailscale任意 VPNtailscale.com/downloads

共通準備

cd ~/Desktop/oiteru_202603
git pull
sudo apt update
sudo apt install -y git tmux python3-full python3-venv python3-pip mysql-server curl

親機

cp .env.example .env
nano .env
scripts/setup_local_mysql.sh
scripts/tmux_oiteru.sh start parent
scripts/tmux_oiteru.sh attach parent

管理画面は http://<親機IP>:5000/admin です。

子機

cp config.example.json config.json
nano config.json
scripts/tmux_oiteru.sh start unit
scripts/tmux_oiteru.sh attach unit

SERVER_URL, UNIT_NAME, UNIT_PASSWORD を子機ごとに設定してください。

tmux 操作

やりたいことコマンド
一時退出Ctrl+b の後に d
状態確認scripts/tmux_oiteru.sh status
親機に戻るscripts/tmux_oiteru.sh attach parent
子機に戻るscripts/tmux_oiteru.sh attach unit
ログを見るscripts/tmux_oiteru.sh logs parent
停止scripts/tmux_oiteru.sh stop parent

確認コマンド

git branch --show-current
git status --short
tmux ls
systemctl status mysql
curl http://localhost:5000
