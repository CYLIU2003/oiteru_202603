# OITERU 取扱説明書

このフォルダには、`OITERU` の使い方をまとめています。

初心者の方は、まず `取説書/QUICKSTART.md` を読んでください。
HTMLで見たい場合は `取説書/QUICKSTART.html` をブラウザで開いてください。

## OITERUとは

OITERU は、ICカードをかざすと生理用品を受け取れるシステムです。

システムは大きく 3 つの役割に分かれます。

| 役割 | 説明 |
| --- | --- |
| 親機 | メインのサーバーです。利用履歴や設定を管理します。 |
| 従親機 | 親機のデータを使うサブサーバーです。必要な拠点で使います。 |
| 子機 | Raspberry Pi で動く端末です。カード読み取りと排出を担当します。 |

## 補足

子機は `unit.py` が実体の入口ですが、中身の本体処理は `archive/unit_client.py` にあります。

そのため、子機の動きが気になるときは次の順で見ると理解しやすいです。

1. `config.json`
2. `unit.py`
3. `archive/unit_client.py`
