## 🎯 OITERUって何？

ICカード（社員証・学生証など）をかざすと、**生理用ナプキンが自動で提供される**プライバシー配慮型の配布システムです。

### 💡 システムの特長

- 🔒 **プライバシー保護**: 誰にも見られずに受け取れる
- 📱 **簡単操作**: ICカードをかざすだけ
- 📊 **在庫管理**: 個人ごとの配布数を自動管理
- 🌐 **複数拠点対応**: 学校や職場の複数箇所に設置可能

```
   ┌─────────────┐                ┌────────────┐
   │   ICカード   │  ───ピッ───▶  │   子機     │  ───▶  🩹 ナプキン！
   │  (社員証等)  │                │ (ラズパイ) │
   └─────────────┘                └─────┬──────┘
                                        │
                                        ▼ 通信
                                  ┌────────────┐
                                  │   親機     │  📊 利用履歴を管理
                                  │ (サーバー) │  🔒 プライバシー保護
                                  └────────────┘
```

## 🔒 プライバシーとセキュリティ

本システムは以下の点でプライバシーに配慮しています：

- 🔐 **匿名性**: 利用履歴は個人が特定できない形で記録
- 🛡️ **アクセス制御**: 管理画面はパスワード保護
- 🗑️ **データ最小化**: 必要最小限の情報のみ保持
- 📊 **統計情報のみ**: 個人の利用パターンは追跡しない
