# 🚨 今すぐカードリーダーを修正する手順

## 現在の状況

Dockerコンテナ内でカードリーダーが検知されていません。

---

## ⚡ 即座に修正（5分で完了）

### 手順1: PowerShellを管理者権限で開く

**Windows キーを押して「PowerShell」と入力**
↓
**右クリック → 「管理者として実行」**

---

### 手順2: 以下のコマンドをコピー&ペースト

```powershell
# USBデバイス一覧を表示
usbipd list
```

**出力例:**
```
BUSID  VID:PID    DEVICE                                    STATE
1-4    054c:0268  Sony FeliCa Port/PaSoRi                  Not shared
```

**カードリーダーのBUSID（例: 1-4）を確認してください**

---

### 手順3: USBをWSLにアタッチ

**BUSIDを確認したら、以下のコマンドを実行:**

```powershell
# 例: BUSIDが1-4の場合
usbipd bind --busid 1-4
usbipd attach --wsl --busid 1-4
```

**⚠️ BUSIDは環境によって異なります。必ず手順2で確認したBUSIDを使用してください。**

成功すると以下のようなメッセージが表示されます:
```
usbipd: info: Using WSL distribution 'Ubuntu' to attach
```

---

### 手順4: Dockerコンテナを再起動

**PowerShellで以下を実行:**

```powershell
wsl -d Ubuntu -e bash -c "cd /mnt/c/Users/Owner/Desktop/oiteru_250827_restAPI && docker-compose restart"
```

または

**WSLターミナルで以下を実行:**

```bash
cd /mnt/c/Users/Owner/Desktop/oiteru_250827_restAPI
docker-compose restart
```

---

### 手順5: 確認

```bash
# WSLターミナルで実行
cd /mnt/c/Users/Owner/Desktop/oiteru_250827_restAPI
python3 diagnostics.py summary
```

**期待される出力:**
```
[  OK  ] NFCリーダー        3個のUSBデバイスを検出
```

または

```bash
# コンテナ内で直接確認
docker exec oiteru_flask ls -la /dev/bus/usb/
```

デバイスファイルが表示されればOKです。

## ❌ usbipd がない場合

### インストール方法

```powershell
# PowerShell（管理者）で実行
winget install usbipd
```

インストール後、PowerShellを再起動してください。

## 🔍 トラブルシューティング

### エラー: "Access is denied"
→ PowerShellを管理者権限で開いているか確認

### エラー: "usbipd: command not found"
→ usbipd-winをインストール: `winget install usbipd`

### カードリーダーが見つからない
→ カードリーダーがPCに物理的に接続されているか確認

### WSLが起動しない
→ PowerShellで `wsl` を実行してWSLを起動
