システム設定

設定は自動的に全ての親機・従親機・子機に同期されます。

自動登録モード設定
有効にすると、未登録のカードがタッチされた際に自動でユーザー登録されます。 事前登録なしでシステムを運用開始できます。
自動登録されたユーザーに付与される初期の取得可能数です。

指定した期間内に取得できる最大数を設定します。期間が変わると上限はリセットされます。

個まで
📌 現在の設定: {% if settings.limit_period == 'day' %}1日{% elif settings.limit_period == 'week' %}1週間{% else %}1ヶ月{% endif %}あたり {{ settings.usage_limit }}個まで取得可能
リセットのタイミング:
・1日: 毎日0時にリセット
・1週間: 毎週月曜日0時にリセット
・1ヶ月: 毎月1日0時にリセット
設定バージョン: {{ settings_version }}
設定の同期について

設定は以下のように同期されます：

  • 親機間: データベースを通じて自動同期
  • 子機へ: ハートビート通信で自動同期（約3秒ごと）

設定を変更すると、次のハートビートで全ての子機に反映されます。
